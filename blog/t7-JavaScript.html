Ruby Vs. JavaScript

January 16, 2015

This was week 1 of the JavaScript Phase 0 track, and after a month or so of using Ruby, I was a bit nervous about having to learn a new language so soon. After a few days though, I'm happy to say that I'm picking up JavaScript relatively well. It was very comforting for me to realize, that though there are certainly syntactical differences between the two, programming is programming at the end of the day. If you can apply logic aptly to one language, you can most likely do so in another. In this post, we'll go over looping and iterations.
