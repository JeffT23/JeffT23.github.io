Technical Blog: The #group_by Method

December 14, 2014

I want to do a brief write-up for you about the #group_by method in the Enumerable module of Ruby. Simply put, the #group_by method breaks a collection into groups based on given criteria. First, the user inputs a block containing the criteria by which the items in the collection will be grouped. Group_by then returns a hash where each key represents a group. Each key’s corresponding value is an array containing the members of that group.

This is how #group_by works for an array of numbers (example courtesy of ruby-doc.org):

(1..6).group_by { |i| i%3 }
=> {0=>[3, 6], 1=>[1, 4], 2=>[2, 5]}

As you can see, each index key represents a subgroup of the original array; all 6 numbers are accounted for among the 3 keys. Let’s now see how #group_by can potentially work for a string array:

["Bonds", "McGwire", "Sosa", "Palmeiro", "A-Rod", "Clemens", "Pettitte", "Cabrera", "Giambi", "Cruz", "Canseco", "Bartolo", "Anderson", "Braun"].group_by { |s| s[0] }

=> {"B"=>["Bonds", "Bartolo", "Braun"], "M"=>["McGwire"], "S"=>["Sosa"], "P"=>["Palmeiro", "Pettitte"], "A"=>["A-Rod", "Anderson"], "C"=>["Clemens", "Cabrera", "Cruz", "Canseco"], "G"=>["Giambi"]}

For this exercise, I input a list of MLB players (extra credit if you see the correlation in the names) and I asked #group_by to return them grouped by the first letter in their names. The first letter is the key, and the associated values are the newly organized names that start with that key. Here's another one where I group my cohort buddy's names by name length.

["Jeffrey", "Christopher","Dan","Max", "Natalia", "Trevor", "Leo"].group_by{|name| name.length}
=> {7=>["Jeffrey", "Natalia"], 11=>["Christopher"], 3=>["Dan", "Max", "Leo"], 6=>["Trevor"]}

There are a plethora of ways that we can organize and group data using this method. The important thing to do is familiarize yourself with the format of #group_by and practice incorporating it within your own code. I highly recommend using http://ruby-doc.org/ when learning to utilize methods such as this one.
