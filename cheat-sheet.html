Tag Description
<!DOCTYPE html > document type: defines specs the document follows
<html> required at top of html file
<head> contains information about the file
<body> contains actual contents of file
<h1> to <h6> different sized headings with h1 being the largest
<p> paragraph
<ol> ordered list
<ul> unordered list
<li> content within ordered or unordered lists
<table> initiates a table within your file
<thead> table header
<tbody> body of table
<tr> needed for every row in your table
<td> needed for every column in your table
<colspan> sets the width of your columns, default is 1
<a> used to set a link to another page or website
<img> links to an image
